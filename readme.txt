-----------------------------------------------------------------------
      AviUtl 0.98系専用 周期間引きプラグイン ver0.01  by MakKi
-----------------------------------------------------------------------

【機能】

  AviUtlが保持している24fps周期に従って間引きフレームを決定します。
  ノンインターレースの24fpsの映像を一定周期で間引きたい時などに有効です。

  0.97系までは標準で実装されているので必要ありません。
  0.99ではAviUtl本体が24fps周期を持っていないため、使用できません。

【インストール】

  delcycle.auf を AviUtl の認識可能なフォルダにコピーしてください。

【使用方法】

  設定→インターレースの解除 から周期間引きを選択することで有効になります。
  設定ウィンドウのタイトルバーに現在のフレームの周期が表示されます。
  編集→24fps周期の編集 で間引くフレームが合うように調節してください。
  （デフォルトではInsert・Deleteキーにショートカットが割り当てられています）

  ○パラメータ

    ・周期4を間引くようにする チェック
          周期0でなく4を間引きます。0.97系にあったものを再現しただけで、特に意味はありません。
